# Engineering standards

This repo follows the **Petrarca Labs** org-wide engineering standards in
[`../engineering-handbook`](../engineering-handbook/INDEX.md) (a sibling repo on
disk; private: https://github.com/petrarca/engineering-handbook).

Start at [`../engineering-handbook/INDEX.md`](../engineering-handbook/INDEX.md).

**For AI coding agents:** when changing releases, tests, dependencies, Taskfile
tasks, commit messages, or tsconfig/`SONNET_UI_LOCAL`, read the matching standard
under `../engineering-handbook/standards/` and prefer it over local defaults.
